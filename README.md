# 🔥 Fire Detection and Suppression Robot

## 📌 Overview
Autonomous robot that detects fire using IR sensors and extinguishes it using a water pump system.

---

## 🎯 Features
- Fire detection (Left, Front, Right)
- Autonomous movement
- Automatic water spray
- Servo-based directional control

---

## 🛠️ Components
- Arduino UNO
- L298N Motor Driver
- IR Sensors
- Servo Motor
- Water Pump

---

## ⚙️ Working
1. Sensors detect fire
2. Robot moves toward fire
3. Stops and activates pump
4. Servo sprays water
